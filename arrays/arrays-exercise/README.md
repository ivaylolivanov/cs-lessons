# Масиви - Упражнение

## Задача

Въведете размер `N` и после `N` цели числа. Преди въвеждането на числата, въведете и едно търсено число.

Програмата трябва да:
1. Намери и изведе дали търсеното число се среща в масива (`"Number found"` или нищо)
2. Изведе средноаритметичното на всички числа
3. Изведе сумата на всички числа
4. Изведе максималното и минималното число на един ред
5. Завърти масива наляво с 1 позиция (първият елемент отива накрая)
6. Изведе завъртения масив с `", "` между елементите

## Примерни операции

``` c#
// Намиране на максимум и минимум по време на четене
int max = int.MinValue;
int min = int.MaxValue;
// ...
if (array[i] > max) max = array[i];
if (array[i] < min) min = array[i];

// Завъртане наляво с 1 позиция
int firstNumber = array[0];
for (int i = 1; i < array.Length; i++)
{
    array[i - 1] = array[i];
}
array[array.Length - 1] = firstNumber;
```
