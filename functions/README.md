# Функции

## Защо?
До момента сме прилагали операции само върху стойности. В процес на разработване на решения обаче виждаме, че има не само стойности, които се повтарят, а дори цели процедури, които са шаблонни. С други думи, в програмирането ни трябва механизъм, с който можем да групираме инструкции, но да ги прилагаме върху различни входни данни. Това са функциите. Можем да мислим, че докато променливите съхраняват стойности или данни, функциите съхраняват действия.

## Какво са функциите
Функциите групират набор от действия, които се прилагат върху набор от данни, може да са входни, а след това може да има изходни данни.

## Как се създава
Създаването на функции има следната структура:

``` c#
тип Име(Параметри, изредени, със, запетая)
{
    Тяло на функцията
    ...
}
```

Възможно е да няма параметри, тогава просто кръглите скоби се оставят празни

``` c#
тип Име()
{
    Тяло на функцията
    ...
}
```

Пример за функция с един параметър, която сумира числата от 0 до N и връща резултата:

``` c#
float SumUpTo(int n)
{
    flaot sum = 0;
    for (int i = 0; i <= n; i++)
    {
        sum += i;
    }

    return sum;
}
```

Пример за функция с един параметър, която не връща резултат и има 2 параметъра:

``` c#
void PrintBetween(int start, int end)
{
    for (int i = start; i <= end; i++)
    {
        Console.WriteLine(i);
    }
}
```

## Употреба
След като сме описали някаква функция, то най-вероятно на нас ни трябва да я употребим. Както при променливите, така и при функциите - използваме името.
Тук обаче имаме и допълнение - добавяме кръглите скоби, в които описваме параметрите. Ако използваме примера от по-горе за сумата на първите N числа, то извикването на
фукцията би изглеждало така:

``` c#
    float sumUpTo17 = sumUpTo(17);
    Console.WriteLine(sumUpTo(31));
```

Това ще събере числата от 0 до 10 включително. В този случай, функцията връща резултат и ако искаме да го използваме, то трябва да го запазим или да го
подадем директно като параметър на друга функция както е в примера по-горе с "Console.WriteLine();".
