# Разработване на игри

## Какво е MonoGame?

MonoGame е безплатен фреймуърк с отворен код за разработване на игри в C#. Позволява създаване на 2D и 3D игри, които работят на Windows, macOS, Linux, Android и iOS.

## Game Loop

Всяка игра работи на базата на т. нар. **game loop** - непрекъснато повтарящ се процес от 3 стъпки:

1. **Обработка на вход** - четене на клавиатура, мишка, геймпад
2. **Обновяване на логиката** - движение, физика, сблъсъци, правила на играта
3. **Рисуване** - изобразяване на текущото състояние на екрана

В MonoGame тези стъпки се реализират в два метода на класа `Game`:

``` c#
protected override void Update(GameTime gameTime)
{
    // Обработва вход и обновява логиката
}

protected override void Draw(GameTime gameTime)
{
    // Рисува всичко на екрана
}
```

## Координатна система

В MonoGame (и повечето графични системи) координатната система е:
- Началото `(0, 0)` е в горния ляв ъгъл
- `X` расте надясно
- `Y` расте надолу

## Основни типове

| Тип | Описание |
|-----|----------|
| `Vector2` | Двумерен вектор с X и Y координати |
| `Rectangle` | Правоъгълник с позиция и размер |
| `Texture2D` | Двумерна текстура (изображение) |
| `SpriteBatch` | Групира и рисува спрайтове ефективно |
| `Color` | Цвят |

``` c#
protected override void Initialize()
{
    // Задаване на начални стойности на променливи
    base.Initialize();
}

protected override void LoadContent()
{
    // Зареждане на текстури, звуци и шрифтове
}
```
