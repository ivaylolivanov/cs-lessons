# Решения от изпит - Масиви

## Задача: Ляво завъртане с 2 позиции

Въведете брой `N` и после `N` цели числа. Завъртете масива наляво с 2 позиции (първите два елемента отиват накрая). Изведете резултата.

**Пример:** `[1, 2, 3, 4, 5]` → `[3, 4, 5, 1, 2]`

Решението предлага два подхода:
- **Решение 1:** Запазване на двата елемента наведнъж и преместване на останалите
- **Решение 2:** Повтаряне на еднократното завъртане два пъти с цикъл

## Задача: Само нечетните числа

Въведете брой `N` и после `N` цели числа. Изведете само нечетните числа, всяко на нов ред. Ако няма нечетни числа, изведете `"No Odds!"`.
