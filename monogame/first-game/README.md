# Първа игра

## Описание

Основен прототип на платформър. Правоъгълник (играч) може да се движи наляво и надясно, а гравитацията го привлича към земята.

## Механики

- **Движение:** Клавиши `A` (наляво) и `D` (надясно) местят играча по X-ос
- **Гравитация:** Играчът пада надолу по Y-ос докато не достигне земята
- **Земя:** Червен правоъгълник в долната лява част на екрана

## Структура на кода

### Game1.cs - Главен клас на играта

Наследява `Game` и съдържа игровия цикъл.

| Поле            | Тип                   | Описание                                     |
|-----------------|-----------------------|----------------------------------------------|
| _graphics       | GraphicsDeviceManager | Управлява графичните настройки               |
| _spriteBatch    | SpriteBatch           | Рисува спрайтове                             |
| _squareTexture  | Texture2D             | Текстура 1x1 пиксел (оцветява правоъгълници) |
| _playerPosition | Vector2               | Позиция на играча                            |
| _playerSize     | Vector2               | Размер на играча (40x65)                     |
| _ground         | float                 | Y-координата на земята (400)                 |

### Player.cs - Структура на играча

Засега празна структура, подготвена за бъдеща логика на играча.
