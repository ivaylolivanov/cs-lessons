# Морска битка - 1D

## Описание

Едноизмерна версия на играта "Морска битка", реализирана с масив от символи. Играчът трябва да открие всички кораби, скрити в морето.

## Правила

- Морето е масив с произволна дължина между 10 и 20 клетки
- Корабите (`'I'`) се поставят на случайни позиции с вероятност 30%, но не повече от 3 броя
- Водата се представя с `'~'`
- При показване на полето корабите са скрити - изглеждат като вода
- Играчът въвежда индекс (позиция) за гадаене
- При улучен кораб клетката се маркира с `'X'` и броят кораби намалява
- Играта свършва когато всички кораби са потопени

## Символи

| Символ | Значение |
|--------|----------|
| `~` | Вода (видима) |
| `I` | Кораб (скрит при показване) |
| `X` | Потопен кораб |

## Пример за изход

```
~~~~~~~~~~~~~~~~~~~~~~~~~~
(играчът въвежда: 5)
~~~~~X~~~~~~~~~~~~~~~~~~~~
(играчът въвежда: 12)
```
